about=Plugin allows selecting multiple OSM features by clicking on the QGIS canvas.
    It displays their tags in a multi-column comparison table with colour coding.
    New in v4:
    - "Add feature" button in the comparison dialog to add a 3rd, 4th, ... feature column.
    - Selecting the same OSM element again triggers a version-history comparison
      (current version vs. the immediately preceding version, fetched from the OSM REST API).
    New in v0.6.1.1:
    - Live rubber-band preview in the "Select OSM Feature" dialog: as you move
      through the list, the corresponding feature is highlighted on the canvas
      so you can visually identify it before confirming your choice.
    New in v0.6.1:
    - Selected features are highlighted on the QGIS canvas with a colour-coded
      rubber band (matching the column colour in the comparison table).
      Highlights are cleared when a new comparison session starts.
    New in v5:
    - Closing the history comparison dialog now fully deactivates the plugin
      (previously the tool remained active waiting for the next canvas click).
    Colour coding:
    - green:           identical value across all compared items
    - yellow:          key present in all, but values differ
    - feature colour:  key unique to one feature / version
    - light gray:      key absent in this feature
